# Tetris
Tetris Game

## Installation

```
$ g++ -c game.cpp && g++ game.o -o tetris -lsfml-graphics -lsfml-window -lsfml-system
```
You will find a binary file named tetris. Just run - 

```
$ ./tetris
```
Now Enjoy ! :D
